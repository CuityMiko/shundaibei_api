# 顺带呗微信小程序项目api服务

## 项目启动
+ 1. 默认端口号为3001，可在config/site_conf文件下修改port字段
+ 2. 项目运行
```bash

# git clone
git clone https://git.qcloud.com/shundaibei/sdb-wxapp-api.git

# install
cd sdb-wxapp-api
npm install

# install node-dev
npm install node-dev -g

# run
npm start

# browser
http://localhost:3001/
```
